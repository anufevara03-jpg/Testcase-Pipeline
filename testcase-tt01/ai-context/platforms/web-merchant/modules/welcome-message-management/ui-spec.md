### 顶部指标区

- **展示内容**：欢迎语数、启用欢迎语、禁用欢迎语、分组数量
- **刷新规则**：指标不随平台、分组和搜索筛选变化
- **颜色标识**：
  - 启用：绿色
  - 禁用：灰色

### 分组管理区

- **左侧分组导航**：
  - 支持分组名称模糊搜索
  - 分组列表展示，默认分组置顶
  - 分组名称长度限制20字符
  - 添加分组按钮
- **分组操作**：
  - 新增分组：弹窗输入分组名称
  - 编辑分组：修改分组名称
  - 删除分组：二次确认后删除，内容移至默认分组

### 筛选与结果列表区

**查询条件**：
- 欢迎语标题：文本输入框，支持模糊查询
- 发送设置：下拉选择框（全部/原文发送/翻译后发送）
- 激活码：级联多选（当选择部分激活码时显示）

**结果列表列**：
1. **勾选列**：支持当前页多选，用于批量操作
2. **欢迎语标题**：展示规则名称，过长省略
3. **适用范围**：
   - 全部激活码：显示"全部"
   - 部分激活码：显示"部分激活码"，悬浮显示具体激活码
4. **欢迎语内容**：展示素材摘要，文本内容显示前20字符，图片显示[图片]标识
5. **发送设置**：显示"原文发送"或"翻译后发送"
6. **延时间隔**：显示"2s-5s"格式
7. **状态**：启用/禁用，使用开关组件
8. **操作列**：编辑、移动分组、删除按钮

## 操作流程

### 新增/编辑欢迎语

**弹窗布局**：
```
┌─────────────────────────────────────────┐
│ 新增欢迎语 / 编辑欢迎语                  │
├─────────────────────────────────────────┤
│ 欢迎语标题: [________________] *         │
│          (最多20字符)                    │
│                                         │
│ 话术分组: [▼] *                        │
│          (下拉选择，默认当前分组)         │
│                                         │
│ 发送设置: ○ 原文发送                    │
│          ● 翻译后发送 *                  │
│                                         │
│ 延时间隔: [2s]----[5s] *               │
│          (范围滑块，默认2-5秒)           │
│                                         │
│ 适用范围: ○ 全部激活码                  │
│          ● 部分激活码 *                  │
│          激活码: [▼▼▼]                 │
│          (多选框，至少选一个)            │
│                                         │
│ 欢迎语内容:                             │
│  ┌─────────────────────────────────┐     │
│  │ [添加素材]                    │     │
│  │                                │     │
│  │ 1. [素材1] [删除]              │     │
│  │ 2. [素材2] [删除]              │     │
│  │                                │     │
│  │ 最多添加2条内容                │     │
│  └─────────────────────────────────┘     │
│                                         │
│ [取消] [保存]                           │
└─────────────────────────────────────────┘
```

**校验规则**：
- 必填字段标 * 号
- 分组名称重复时提示"分组名称已存在"
- 欢迎语内容超过2条时提示"欢迎语内容最多2条"
- 未选择激活码时提示"请选择适用激活码"

## 特殊场景处理

### 跨端预览

- 在配置页面右侧或新标签页展示PC端预览效果
- 预览内容按平台和激活码适用范围展示
- 实时更新，配置变更后即时刷新预览

### 状态管理

- 启用状态：绿色开关，显示"启用"
- 禁用状态：灰色开关，显示"禁用"
- 禁用后的欢迎语不在PC端显示和触发

### 异常提示

- 分组名称重复：红色文字提示"分组名称已存在"
- 字段为空：红色文字提示"请输入XXX"
- 文件格式错误：红色文字提示"不支持的文件格式"
- 文件大小超限：红色文字提示"文件大小不能超过10MB"
- 操作失败：Toast提示具体失败原因
