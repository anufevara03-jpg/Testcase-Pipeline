# welcome-message-management 需求说明

## 用户故事

- 作为商家客服管理员，我希望能够配置欢迎语，让新客户首次联系时收到自动回复
- 作为商家管理员，我希望能够按不同社交平台配置不同的欢迎语
- 作为商家管理员，我希望能够启用或禁用特定的欢迎语配置
- 作为商家客服，我希望在PC端看到当前配置的公共欢迎语

## 验收标准

- [ ] 欢迎语按社交平台维护，平台切换后刷新平台下的话术分组和规则列表
- [ ] PC端-欢迎语-预览处按平台和当前激活码适用范围显示当前平台的主管端欢迎语
- [ ] 新增商户时，系统生成默认分组，默认分组不允许编辑、删除
- [ ] 分组名称最长20字符，不能重复，重复提示"分组名称已存在"
- [ ] 欢迎语标题最长20字符，必填
- [ ] 一套欢迎语最多配置2条回复内容
- [ ] 发送设置支持"原文发送"和"翻译后发送"两种方式
- [ ] 延时间隔支持2-5秒范围配置
- [ ] 适用范围支持"全部激活码"和"部分激活码"选择
- [ ] 删除分组时，将该分组下的内容移动到默认分组
- [ ] 删除欢迎语后，自动删除快捷回复、欢迎语、关键词回复中引用的素材内容

## 业务规则

### 顶部指标区业务规则

| 指标 | 计算口径 | 说明 |
| :--- | :--- | :--- |
| **欢迎语数** | 当前商户全部欢迎语数量 | 不随平台、分组和搜索筛选变化 |
| **启用欢迎语** | 状态为启用的欢迎语数量 | 只有启用的数据才允许下发到 PC 欢迎语-规则预览 |
| **禁用欢迎语** | 状态为禁用的欢迎语数量 | 禁用的数据不显示在 PC 欢迎语-规则预览 |
| **分组数量** | 话术分组数量 | 默认分组计入 |

### 页面联动规则

- 欢迎语、关键词、快捷回复的后台配置属于主管端公共配置，按平台、适用激活码、状态下发到 PC 右侧快捷回复的公共 Tab
- PC 端可追加个人配置
- 公共快捷回复用于人工快捷发送
- 公共的欢迎语和关键词回复如满足条件会自动触发
- 回复内容来源于话术素材库。素材库更新内容或名称后，欢迎语、关键词回复和公共快捷回复中的引用展示应同步更新
- 素材库删除素材后，应自动移除引用

### 自动回复功能定位与优先级

| 自动回复功能 | 定位 | 功能内部优先级 |
| :--- | :--- | :--- |
| **PC-右侧自动回复** | 非首聊时触发，该功能仅能在 PC 端维护，为 PC 端联系人维度的自动回复，一个联系人仅能维护一条，且只要命中就会触发 | 命中即触发 |
| **PC/后台系统-欢迎语自动回复** | 仅首聊时触发，PC 端维护的是个人的欢迎语，后台维护的是公共的欢迎语 | 1. 触发后随机选取一套已启用欢迎语模板发送<br>2. 如果一套模板内多条内容，按添加顺序发送 |
| **PC/后台系统-关键词自动回复** | 首聊和非首聊都会触发，PC 端维护的是个人的关键词，后台维护的是公共的关键词 | 1. 多套同时命中时，仅回复一套<br>2. 匹配优先级：全匹配 > 半匹配，同时触发全匹配和半匹配情况下仅发送全匹配关键词回复内容<br>3. 多个半匹配同时命中时：关键词最长 > 长度相同随机 |

### 特殊场景触发规则

| 首聊/非首聊 | 场景 | 触发规则 |
| :--- | :--- | :--- |
| **首聊** | 同时命中欢迎语和关键词 | 欢迎语 > 关键词回复，如果欢迎语命中，则关键词回复不触发 |
| **非首聊** | 同时命中关键词和 PC 右侧-自动回复 | 只要命中，二者都触发 |

### 新增/编辑分组规则

| 字段 | 类型 | 是否必填 | 校验/说明 |
| :--- | :--- | :--- | :--- |
| **分组名称** | 文本输入 | 是 | 1. 最长 20 个字符<br>2. 不能与当前商户下的已有欢迎语分组重复，重复提示"分组名称已存在" |

**保存时需校验：**

- 分组名称不能为空，为空提示"请输入分组名称"
- 分组名称不能超过 20 个字符
- 分组名称不能与当前商户下的已有分组重复，新增/编辑时都需要做唯一性校验，重复提示"分组名称已存在"

### 删除分组规则

- 删除分组前需弹窗让用户二次确认：确认删除该分组吗？删除后该分组下的内容将会移动到默认分组
- 删除分组后，将当前分组下的内容移动到默认分组

### 新增/编辑欢迎语规则

| 字段 | 类型 | 是否必填 | 校验/说明 |
| :--- | :--- | :--- | :--- |
| **欢迎语标题** | 文本输入 | 是 | 最长 20 个字符；用于列表、PC 端预览和规则识别，可以重复 |
| **话术分组** | 下拉选择 | 是 | 默认选择当前分组，新增后归属该分组 |
| **发送设置** | 单选 | 是 | 1. 原文发送：直接发送素材内容<br>2. 翻译后发送：按 PC 客户端当前聊天对象的翻译设置自动翻译后发送 |
| **延时间隔** | 范围滑块 | 是 | 客户首条消息触发欢迎语后，在配置的时间区间内随机取值并延迟发送；默认 2s-5s，支持手动调整 |
| **适用范围** | 单选+激活码级联 | 是 | 1. 全部激活码表示该平台全部 PC 客服端可见、可触发<br>2. 部分激活码需选择具体激活码，未命中的激活码不下发该欢迎语 |
| **欢迎语内容** | 素材选择 | 是 | 从话术素材库选择文本、图片等素材，支持选择多条并保存顺序；一套欢迎语最多配置 2 条回复内容 |

**保存时需校验：**

- 欢迎语标题、话术分组、发送设置、延时间隔、适用范围、欢迎语内容不能为空；为空时提示"请输入/请选择xxx"
- 选择部分激活码时，至少选择 1 个激活码；未选择时提示"请选择适用激活码"
- 欢迎语内容最多 2 条，超过上限时禁止继续选择并提示"欢迎语内容最多 2 条"
- 保存成功后关闭弹窗，刷新当前列表并保留当前平台、分组和查询条件

### 删除/批量删除规则

| 操作 | 提示文案 | 提交结果 |
| :--- | :--- | :--- |
| **单条删除** | 确认删除该记录吗？删除后该记录将不再使用和展示所选内容 | 删除当前内容，成功后刷新列表 |
| **批量删除** | 至少选择 1 条数据，未选择时按钮禁用且悬浮时提示"至少选中一项数据"<br>确认文案：确认删除该记录吗？删除后该记录将不再使用和展示所选内容 | 删除全部已选内容，成功后刷新列表并清空已选状态 |
