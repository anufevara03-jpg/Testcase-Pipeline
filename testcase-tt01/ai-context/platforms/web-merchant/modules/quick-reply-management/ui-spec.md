# 快捷回复管理 UI 交互说明

> 来源：快捷回复管理模块.md §一、§二、§三、§四、§五

## 页面列表

- 快捷回复管理列表页（路径待补充）

## 页面布局

- 顶部指标区：快捷回复数 / 启用回复 / 禁用回复 / 分组数量（源：§一.1）
- 平台筛选区：按社交平台（WhatsApp/Telegram/LINE 等）维护，展示平台下快捷回复数量（源：§二）
- 左侧：话术分组导航（支持分组名模糊搜索、新建/编辑/删除分组）（源：§三）
- 筛选区：关键字（标题/回复内容/分组名模糊）、激活码（级联多选，可选激活码分组或单个激活码）（源：§四.1）
- 列表列：勾选 / 标题 / 适用范围 / 回复内容 / 创建人·创建时间 / 状态 / 操作（源：§四.2）

## 页面跳转逻辑

- 点"添加快捷回复"/编辑 → 打开新增/编辑快捷回复弹窗（源：§五.1）
- 平台切换 → 刷新平台下话术分组与快捷回复列表；PC 公共 Tab 展示当前平台启用数据（源：§二）
- 操作：编辑、删除；删除前二次确认（源：§四.2）

## 前端校验规则

| 字段 | 规则 | 触发时机 | 错误提示 | 来源 |
| ---- | ---- | -------- | -------- | ---- |
| 分组名称 | 必填，≤20 字符，商户下不重复 | 提交 | "请输入分组名称" / "分组名称已存在" | §三.1 |
| 标题 | 必填，≤20 字符 | 提交 | "请输入xxx" | §五.1 |
| 话术分组 | 必填，默认当前分组 | 提交 | （空提示文案文档未明确） | §五.1 |
| 适用激活码 | 必填，默认全部；部分至少 1 个 | 提交 | （空提示文案文档未明确） | §五.1 |
| 回复内容 | 必填，选素材，≤5 项 | 提交 | "请输入xxx" | §五.1 |
| 批量删除 | 至少选 1 条 | 点击批量删除 | 按钮禁用 + 悬浮提示"至少选中一项数据" | §五.2 |

## 异常状态展示

- 删除/批量删除：二次确认文案——单条"确认删除该记录吗？删除后将不可恢复且客户端将不再展示所选内容。"；批量"确定删除选中的 N 条记录吗？删除后将不可恢复且客户端将不再展示所选内容。"（源：§五.2）
- 删除分组：二次确认"确认删除该分组吗？删除后该分组下的内容将会移动到默认分组。"（源：§三.2）
