## 回归测试重点

- 线索去重范围 6 种切换的正确性（当前工单 / 含底库 / 账号下全部 / 关联账号 / 分组内 / 指定激活码），跨平台隔离（源：模块文档 §二.4、§四.5）
- 端口分配动态/固定校验：动态可用数 = 端口总数 − 固定分配总数 − 动态分配在线社交账号数（源：§四.1）
- 启用/停用联动：停用 → PC 强制下线 + 回收端口 + 工单置禁用（源：§五.2）
- 多设备登录关闭时挤下线（源：§四.8）
- 删除分组后激活码移入默认分组，去重范围"分组内"按新分组去重、历史不回溯（源：§三.3）
- PC 登录前置校验五条件：激活码存在/状态启用/平台已配置/端口有余额/时间符合工作时间（源：§五.6）
