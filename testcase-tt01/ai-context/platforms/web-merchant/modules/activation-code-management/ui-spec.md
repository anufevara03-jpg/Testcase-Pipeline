# 激活码管理 UI 交互说明

> 来源：激活码管理模块.md §一、§三、§四、§五、§六

## 页面列表

- 激活码管理列表页（路径待补充）

## 页面布局

- 顶部：统计卡片（激活码总数 / 启用数 / 停用数 / 分组数）（源：§一、§二.1）
- 左侧：分组导航；右侧：激活码列表（支持多选）（源：§一）
- 筛选条件：激活码(下拉) / 平台(WhatsApp/Telegram/Line) / 状态(全部/启用/停用) / 创建时间(时间段)（源：§一、§六.1）
- 操作按钮：添加激活码 / 批量移动分组 / 批量删除（源：§一）

## 页面跳转逻辑

- 点击分组 → 右侧只展示该分组激活码；点击全部 → 展示全部（源：§三.1）
- 启用状态操作列：查看 / 编辑 / 分享 / 移动分组 / 聊天记录（源：§六.2）
- 停用状态操作列：移动分组 / 删除（源：§六.2）
- 登录日志：点击查看激活码登录详情（激活码 / 平台 / 登录时间 / 登录设备 / 登录 IP / 登录地区 / 登录结果）（源：§五.4）
- 分享工单：同平台工单-分享工单功能，权限由分享页面权限控制（源：§五.5）

## 前端校验规则

| 字段 | 规则 | 触发时机 | 错误提示 |
| ---- | ---- | -------- | -------- |
| 分组名称 | 必填，最长 20 字符，同商户不可重复 | 提交 | "分组名称已存在"（源：§三.2） |
| 必填项 | 端口分配/分组/社交平台/去重范围/工作时间/多设备登录/聊天备份/画像共享/数据脱敏/置零时间等必填 | 提交 | "请输入xxx，请选择xxx"（源：§五.1） |
| 固定分配端口数 | >0 | 提交 | 待补充（源：§四.1） |
| 社交平台 | 至少选一个，默认全选 | 提交 | 待补充（源：§四.4） |
| 备注 | 最多 50 字 | 输入 | 待补充（源：§四.14） |
| 批量操作 | 至少选中 1 条 | 点击批量按钮 | 按钮禁用 + 悬浮提示"至少选中一项数据"（源：§五.3） |

## 异常状态展示（PC 登录场景提示文案）

- 激活码不存在：提示"激活码不存在"（源：§五.6）
- 停用激活码：PC 不可新登录，已登录强制下线，提示"该激活码已停用或删除"（源：§五.2、§五.6）
- 端口不足：提示"当前端口数已达上限，请提升端口数或升级套餐"（源：§五.6）
- 多设备登录关闭：新设备登录挤下旧设备，提示"此激活码已在新设备登录"（源：§四.8、§五.6）
- 最近登录时间暂无：显示"-"（源：§六.2）
