# 话术素材库 UI 交互说明

> 来源：话术素材库模块.md §一、§三、§四、§五

## 页面列表

- 话术素材库列表页（路径待补充）

## 页面布局

- 顶部指标区：素材总数 / 文本素材 / 图片素材 / 素材分组（源：§一、§二.1）
- 左侧：分组导航（支持分组名模糊搜索、新建分组）（源：§一、§三）
- 素材分类 Tab：文本 / 图片 / 名片（源：§一）
- 筛选：素材标题模糊查询（源：§四.1）
- 列表列：勾选 / 素材标题 / 素材内容 / 创建人 / 创建时间 / 操作（源：§四.2）

## 页面跳转逻辑

- 文本 Tab → 添加素材（新增文本素材弹窗）/ 导入文本（源：§五.1、§五.3）
- 图片 Tab → 添加素材（直接打开系统文件选择器，支持多选上传）（源：§五.2）
- 图片 Tab 不展示"导入文本"（源：§五.3）
- 操作：编辑 / 移动分组 / 删除；删除前二次确认（源：§四.2）
- 移动分组/批量移动分组：选择目标分组（仅当前素材类型下已有分组）后提交；批量至少选 1 条（源：§五.4）

## 前端校验规则

| 字段 | 规则 | 触发时机 | 错误提示 | 来源 |
| ---- | ---- | -------- | -------- | ---- |
| 分组名称 | 必填，最长 20 字符，同类型不重复 | 提交 | "分组名称已存在" / "请输入分组名称" | §三.2 |
| 素材标题 | 必填，≤20 字符 | 提交 | "请输入素材标题" | §五.1 |
| 素材内容 | 必填，≤2000 字符 | 提交 | "请输入素材内容" | §五.1 |
| 图片文件 | jpg/jpeg/png/gif/webp，单张≤10MB | 选择 | 不允许选择 / 上传按钮置灰 | §五.2 |
| 导入文件 | xls/csv，标题≤20 字 / 内容≤2000 字 | 导入 | 下载失败原因 | §五.3 |
| 目标分组（移动分组） | 必填，只能选当前素材类型下已有分组 | 提交 | （空提示文案文档未明确） | §五.4 |
| 批量操作 | 至少选中 1 条 | 点击批量按钮 | 按钮禁用 + 悬浮提示"至少选中一项数据" | §五.4 |

## 异常状态展示

- 删除素材/批量删除：二次确认文案（单条/批量 N 条），提示删除后将自动清除快捷回复/欢迎语/关键词回复中的引用（源：§五.5）
- 导入失败：任一有效行校验失败则本次整体不入库，支持下载失败原因（源：§五.3）
- 图片格式/大小不符：不允许选择（源：§五.2）
