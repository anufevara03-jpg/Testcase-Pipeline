## 回归测试重点

- 素材总数口径=文本+图片，不随类型/分组/关键词筛选变化（源：模块文档 §二.1）
- 分组名同类型唯一、最长 20 字符（源：§三.2）
- 删除分组 → 组下素材移到默认分组（源：§三.3）
- 删除素材级联清除快捷回复/欢迎语/关键词回复中的引用（源：§五.5）
- 导入校验：任一有效行失败则整体不入库 + 下载失败原因（源：§五.3）
- 图片格式/大小校验：jpg/jpeg/png/gif/webp、单张≤10MB（源：§五.2）
- 素材类型枚举：文字/图片/名片（经用户确认，与 DB 文档五类不一致，见 data-model.md C1）
