# 话术素材库 需求说明

> 端：web-merchant ｜ 模块：material-library
> 主要来源：`TemporaryFile/商户端模块说明/话术素材库模块.md`（下称"模块文档"）

## 用户故事

- 作为商家员工，我希望集中管理文字/图片/名片三类素材，以便在话术回复等模块直接引用（源：模块文档 §一）。
- 作为商家员工，我希望按分组组织素材并支持批量操作，以便高效维护素材库（源：模块文档 §一、§五.4）。
- 作为商家员工，我希望导入文本素材，以便批量录入话术（源：模块文档 §五.3）。

## 验收标准

- [ ] 顶部指标展示素材总数/文本素材/图片素材/素材分组四个指标；素材总数=文本+图片，不随筛选变化（源：模块文档 §二.1）
- [ ] 新增分组名称最长 20 字符，同类型下不可重复，重复提示"分组名称已存在"（源：模块文档 §三.2）
- [ ] 删除分组二次确认，组下素材移到默认分组（源：模块文档 §三.3）
- [ ] 添加文本素材：标题≤20 字符、内容≤2000 字符，为空提示"请输入素材标题/请输入素材内容"（源：模块文档 §五.1）
- [ ] 添加图片素材：支持 jpg/jpeg/png/gif/webp，单张≤10MB；未选时上传按钮置灰（源：模块文档 §五.2）
- [ ] 导入文本：支持 xls/csv，标题≤20 字/内容≤2000 字；任一有效行校验失败则整体不入库，支持下载失败原因（源：模块文档 §五.3）
- [ ] 批量操作未选：按钮禁用，悬浮提示"至少选中一项数据"（源：模块文档 §五.4）
- [ ] 删除素材二次确认，删除后自动清除快捷回复/欢迎语/关键词回复中引用的素材内容（源：模块文档 §五.5）
- [ ] 移动分组/批量移动分组：目标分组为下拉选择，只能选择当前素材类型下已有分组；批量移动至少选 1 条，未选时按钮禁用并悬浮提示"至少选中一项数据"（源：模块文档 §五.4）

## 业务规则

1. 素材总数=文本素材数+图片素材数，不随类型/分组/关键词筛选变化（源：模块文档 §二.1）。
2. 默认分组系统生成，不允许编辑/删除（源：模块文档 §三.2）。
3. 分组名称同类型（文本/图片）下不可重复，最长 20 字符（源：模块文档 §三.2）。
4. 删除分组后组下素材移到默认分组（源：模块文档 §三.3）。
5. 文本素材：标题≤20 字符，内容≤2000 字符（源：模块文档 §五.1）。
6. 图片素材：jpg/jpeg/png/gif/webp，单张≤10MB，名称默认取文件名、≤200 字符（源：模块文档 §五.2）。
7. 导入文本仅文本 Tab 展示，支持 xls/csv；有效行标题/内容不能都为空，都为空行跳过；任一有效行失败则本次不入库（源：模块文档 §五.3）。
8. 删除素材自动清除快捷回复/欢迎语/关键词回复中的引用（源：模块文档 §五.5）。
9. 移动分组/批量移动分组：目标分组为下拉选择、必填，只能选择当前素材类型下已有分组；批量移动至少选 1 条素材，未选时按钮禁用并悬浮提示（源：模块文档 §五.4）。
