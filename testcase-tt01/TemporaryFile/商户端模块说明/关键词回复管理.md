# SCEM项目商户端关键词回复管理（后台关键词回复）模块说明

## 一、 顶部指标区

### 1. 业务规则

| 指标           | 计算口径               | 说明                                                |
| :------------- | :--------------------- | :-------------------------------------------------- |
| **关键词数**   | 当前商户全部关键词数量 | 不随平台、分组和搜索筛选变化。                      |
| **启用关键词** | 状态为启用的关键词数量 | 只有启用的数据才允许下发到 PC 关键词回复-规则预览。 |
| **禁用关键词** | 状态为禁用的关键词数量 | 禁用的数据不显示在 PC 关键词回复-规则预览。         |
| **分组数量**   | 话术分组数量           | 默认分组计入。                                      |

### 2. 后台与PC联动

- 欢迎语、关键词、快捷回复的后台配置属于主管端公共配置，按平台、适用激活码、状态下发到 PC；PC 端可追加个人配置。
- 公共快捷回复用于人工快捷发送；公共的欢迎语和关键词回复如满足条件会自动触发。
- 回复内容来源于话术素材库。素材库更新内容或名称后，欢迎语、关键词回复和公共快捷回复中的引用展示应同步更新；素材库删除素材后，应自动移除引用。

## 二、 平台筛选区

| 项目       | 规则                                                                              |
| :--------- | :-------------------------------------------------------------------------------- |
| **平台**   | 关键词回复按社交平台维护，平台切换后刷新平台下的话术分组和规则列表。              |
| **数量**   | 展示当前平台下的关键词回复数量。                                                  |
| **PC联动** | PC 端-关键词回复-预览处按平台和当前激活码适用范围显示当前平台的主管端关键词回复。 |

## 三、 话术分组管理

### 1. 新增/编辑分组

- **规则**：新增商户时，系统生成默认分组，默认分组不允许编辑、删除。
- **操作说明**：点击添加分组或编辑分组时打开同一个弹窗，弹窗只维护分组名称。

| 字段         | 类型     | 是否必填 | 校验/说明                                                                                                                                               |
| :----------- | :------- | :------- | :------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **分组名称** | 文本输入 | 是       | 1. 最长 20 个字符；<br>2. 不能与当前商户下的已有关键词回复分组重复，重复提示“分组名称已存在”；<br>3. 分组名称变更后，关键词回复列表分组展示应同步变更。 |

**保存时需校验：**

- 分组名称不能为空，为空提示“请输入分组名称”。
- 分组名称不能超过 20 个字符。
- 分组名称不能与当前商户下的已有分组重复，新增/编辑时都需要做唯一性校验，重复提示“分组名称已存在”。

### 2. 删除分组

- 删除分组前需弹窗让用户二次确认：确认删除该分组吗？删除后该分组下的内容将会移动到默认分组。
- 删除分组后，将当前分组下的内容移动到默认分组。

## 四、 查询条件与结果列表

### 1. 查询条件

| 字段/操作        | 类型     | 说明                                 |
| :--------------- | :------- | :----------------------------------- |
| **关键词**       | 文本输入 | 支持按关键词模糊查询。               |
| **发送设置**     | 下拉选择 | 枚举值为全部、原文发送、翻译后发送。 |
| **匹配规则**     | 下拉选择 | 枚举值为全部、全匹配、半匹配。       |
| **激活码(备注)** | 级联多选 | 按适用激活码过滤关键词回复。         |

### 2. 结果列表

| 列/区域       | 规则                                                               |
| :------------ | :----------------------------------------------------------------- |
| **关键词**    | 展示触发关键词。                                                   |
| **适用范围**  | 展示全部激活码或部分激活码；悬浮展示激活码明细。                   |
| **匹配方式**  | 全匹配优先于半匹配；多个半匹配命中时优先最长关键词，长度相同随机。 |
| **回复内容**  | 展示素材摘要；内容来自话术素材库，素材修改后应同步。               |
| **发送设置**  | 原文发送或翻译后发送。                                             |
| **回复方式**  | 回复全部按素材顺序发送；随机回复一条只发送一条。                   |
| **延时间隔**  | 触发后在配置区间内随机延迟发送。                                   |
| **状态**      | 禁用后 PC 不再触发和展示该公共规则。                               |
| **分页/勾选** | 支持多选和批量删除。                                               |

## 五、 操作说明

### 1. 新增/编辑关键词回复

| 字段             | 类型            | 是否必填 | 校验/说明                                                                                                          |
| :--------------- | :-------------- | :------- | :----------------------------------------------------------------------------------------------------------------- |
| **关键词**       | 文本输入        | 是       | 最长 20 个字符；用于匹配客户发送的消息内容，允许重复。                                                             |
| **话术分组**     | 下拉选择        | 是       | 默认选择当前分组，新增后归属该分组。                                                                               |
| **匹配规则**     | 单选            | 是       | 1. 全匹配：客户消息内容与关键词完全一致时触发；<br>2. 半匹配：客户消息内容等于或包含关键词时触发。                 |
| **发送设置**     | 单选            | 是       | 1. 原文发送：直接发送素材内容；<br>2. 翻译后发送：按客户端当前翻译设置自动翻译后发送。                             |
| **延时间隔**     | 范围滑块        | 是       | 消息命中关键词后，在配置的时间区间内随机取值并延迟发送；默认 2s-5s，自定义区间 1s-180s。                           |
| **回复方式**     | 单选            | 是       | 1. 回复全部：按素材添加顺序依次发送全部内容；<br>2. 随机回复一条：从已配置素材中随机发送一条。                     |
| **适用范围**     | 单选+激活码级联 | 是       | 1. 全部激活码表示该平台全部 PC 客服端可见、可触发；<br>2. 部分激活码需选择具体激活码，未命中的客户端不下发该规则。 |
| **自动回复内容** | 素材选择        | 是       | 从话术素材库选择文本、图片等素材，支持选择多条并保存顺序；一个关键词最多配置 5 条回复内容。                        |

**保存时需校验：**

- 关键词、话术分组、匹配规则、发送设置、延时间隔、回复方式、适用范围、自动回复内容不能为空；为空时提示“请输入/请选择xxx”。
- 选择部分激活码时，至少选择 1 个激活码；未选择时提示“请选择适用激活码”。
- 自动回复内容最多 5 条，超过上限时禁止继续选择并提示“关键词回复内容最多 5 条”。
- 保存成功后关闭弹窗，刷新当前列表并保留当前平台、分组和查询条件。

### 2. 删除/批量删除

| 操作         | 提示文案                                                                                                                                  | 提交结果                                                     |
| :----------- | :---------------------------------------------------------------------------------------------------------------------------------------- | :----------------------------------------------------------- |
| **单条删除** | 确认删除该记录吗？删除后该记录将不再使用和展示所选内容。                                                                                  | 删除当前内容；成功后刷新列表。如当前页无数据，则回到上一页。 |
| **批量删除** | 至少选择 1 条数据，未选择时按钮禁用且悬浮时提示“至少选中一项数据”。<br>确认文案：确认删除该记录吗？删除后该记录将不再使用和展示所选内容。 | 删除全部已选内容；成功后刷新列表并清空已选状态。             |

### 3. 自动回复功能定位

- 自动回复包含三类：PC/后台欢迎语回复、PC/后台关键词回复、PC 右侧-自动回复。命中表示：功能已启用、且平台、账号、激活码、关键词、会话状态等条件均满足。

| 自动回复功能                   | 定位                                                                                                             | 功能内部优先级                                                                                                                                                                                                                          |
| :----------------------------- | :--------------------------------------------------------------------------------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **PC-右侧自动回复**            | 非首聊时触发，该功能仅能在 PC 端维护，为 PC 端联系人维度的自动回复，一个联系人仅能维护一条，且只要命中就会触发。 | 命中即触发。                                                                                                                                                                                                                            |
| **PC/后台系统-欢迎语自动回复** | 仅首聊时触发，PC 端维护的是个人的欢迎语，后台维护的是公共的欢迎语。                                              | 1. 触发后随机选取一套已启用欢迎语模板发送；<br>2. 如果一套模板内多条内容，按添加顺序发送。                                                                                                                                              |
| **PC/后台系统-关键词自动回复** | 首聊和非首聊都会触发，PC 端维护的是个人的关键词，后台维护的是公共的关键词。                                      | 1. 多套同时命中时，仅回复一套；<br>2. 匹配优先级：全匹配 > 半匹配，同时触发全匹配和半匹配情况下仅发送全匹配关键词回复内容；<br>3. 多个半匹配同时命中时：关键词最长 > 长度相同随机。优先选取关键词最长的回复内容，如长度相同则随机选取。 |

### 4. 特殊场景

| 首聊/非首聊 | 场景                              | 触发规则                                                  |
| :---------- | :-------------------------------- | :-------------------------------------------------------- |
| **首聊**    | 同时命中欢迎语和关键词            | 欢迎语 > 关键词回复，如果欢迎语命中，则关键词回复不触发。 |
| **非首聊**  | 同时命中关键词和 PC 右侧-自动回复 | 只要命中，二者都触发。                                    |
