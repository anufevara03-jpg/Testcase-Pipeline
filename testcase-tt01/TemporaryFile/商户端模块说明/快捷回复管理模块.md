# SCEM项目商户端快捷回复管理模块说明

## 一、 顶部指标区

### 1. 业务规则

| 指标           | 计算口径                 | 说明                                         |
| :------------- | :----------------------- | :------------------------------------------- |
| **快捷回复数** | 当前商户全部快捷回复数量 | 不随平台、分组和搜索筛选变化。               |
| **启用回复**   | 状态为启用的快捷回复数量 | 只有启用的数据才允许下发到 PC 公共快捷回复。 |
| **禁用回复**   | 状态为禁用的快捷回复数量 | 禁用的数据不显示在 PC 公共快捷回复。         |
| **分组数量**   | 话术分组数量             | 默认分组计入。                               |

### 2. 后台与PC联动

- 欢迎语、关键词、快捷回复的后台配置属于主管端公共配置，按平台、适用激活码、状态下发到 PC 右侧快捷回复的公共 Tab；PC 端可追加个人配置。
- 公共快捷回复用于人工快捷发送；公共的欢迎语和关键词回复如满足条件会自动触发。
- 回复内容来源于话术素材库。素材库更新内容或名称后，欢迎语、关键词回复和公共快捷回复中的引用展示应同步更新；素材库删除素材后，应自动移除引用。

## 二、 平台筛选区

| 项目       | 规则                                                    |
| :--------- | :------------------------------------------------------ |
| **平台**   | 按社交平台维护快捷回复，如 WhatsApp、Telegram、LINE。   |
| **数量**   | 展示当前平台下的快捷回复数量。                          |
| **PC联动** | PC 右侧快捷回复“公共 tab”展示“当前平台”启用的快捷回复。 |

## 三、 话术分组管理

### 1. 新增/编辑分组

- **规则**：新增商户时，系统生成默认分组，默认分组不允许编辑、删除。
- **操作说明**：点击添加分组或编辑分组时打开同一个弹窗，弹窗只维护分组名称。

| 字段         | 类型     | 是否必填 | 校验/说明                                                                                                                                                                          |
| :----------- | :------- | :------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **分组名称** | 文本输入 | 是       | 1. 最长 20 个字符；<br>2. 不能与当前商户下的已有快捷回复分组重复，重复提示“分组名称已存在”；<br>3. 保存后刷新分组列表和当前快捷回复列表，PC 公共快捷回复同步使用新的分组名称展示。 |

**保存时需校验：**

- 分组名称不能为空，为空提示“请输入分组名称”。
- 分组名称不能超过 20 个字符。
- 分组名称不能与当前商户下的已有分组重复，新增/编辑时都需要做唯一性校验，重复提示“分组名称已存在”。

### 2. 删除分组

- 删除分组前需弹窗让用户二次确认：确认删除该分组吗？删除后该分组下的内容将会移动到默认分组。
- 删除分组后，将当前分组下的内容移动到默认分组。

## 四、 查询条件与结果列表

### 1. 查询条件

| 字段/操作  | 类型     | 说明                                                           |
| :--------- | :------- | :------------------------------------------------------------- |
| **关键字** | 文本输入 | 支持按标题、回复内容、分组名称模糊查询。                       |
| **激活码** | 级联多选 | 支持选择激活码分组或单个激活码，筛选指定客服端可用的快捷回复。 |

### 2. 结果列表

| 列/区域             | 规则                                             |
| :------------------ | :----------------------------------------------- |
| **勾选列**          | 支持多选，用于批量删除。                         |
| **标题**            | 展示快捷回复标题。                               |
| **适用范围**        | 展示全部激活码或部分激活码；悬浮展示具体激活码。 |
| **回复内容**        | 展示引用素材名称；悬浮展示素材标题和内容预览。   |
| **创建人/创建时间** | 展示创建人和创建时间。                           |
| **状态**            | 支持启用/禁用；禁用后 PC 公共快捷回复不展示。    |
| **操作**            | 支持编辑、删除；删除前需二次确认。               |

## 五、 操作按钮区

### 1. 新增/编辑快捷回复

| 字段           | 类型     | 是否必填 | 校验/说明                                                                            |
| :------------- | :------- | :------- | :----------------------------------------------------------------------------------- |
| **标题**       | 文本输入 | 是       | 最长 20 个字符，可以重复。                                                           |
| **话术分组**   | 下拉选择 | 是       | 默认选择当前分组，新增后归属该分组。                                                 |
| **适用激活码** | 单选框   | 是       | 默认全部激活码，全部激活码表示该平台全部 PC 客服端可见；部分激活码需选择具体激活码。 |
| **回复内容**   | 素材选择 | 是       | 从话术素材库选择文本或图片素材，支持选择多素材，最多选择 5 项素材。                  |

**保存时需校验：**

- 标题、回复内容不能为空，否则提示“请输入xxx”。
- 保存成功后关闭弹窗，刷新当前文本列表。

### 2. 删除/批量删除

| 操作         | 提示文案                                                                                                                                                 | 提交结果                                         |
| :----------- | :------------------------------------------------------------------------------------------------------------------------------------------------------- | :----------------------------------------------- |
| **单条删除** | 确认删除该记录吗？删除后将不可恢复且客户端将不再展示所选内容。                                                                                           | 删除当前内容，成功后刷新列表。                   |
| **批量删除** | 1. 至少选择 1 条素材；未选择时按钮禁用，鼠标悬浮时提示：至少选中一项数据。<br>2. 确定删除选中的 N 条记录吗？删除后将不可恢复且客户端将不再展示所选内容。 | 删除全部已选内容，成功后刷新列表并清空已选状态。 |
