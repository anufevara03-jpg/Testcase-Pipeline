# SCEM项目商户端话术素材库（话术素材库）模块说明

## 一、 页面概览

- **功能定位**：集中存放文字、图片、名片三类素材。创建一次即可在话术回复等模块直接引用，无需重复准备。
- **顶部指标区**：展示“素材总数”、“文本素材”、“图片素材”、“素材分组”四个核心指标。
- **分组管理**：左侧为分组导航栏，支持按分组名称进行模糊搜索，并支持新建分组。
- **素材分类Tab**：支持按“文本”、“图片”、“名片”分类展示素材。
- **筛选与列表**：支持输入素材标题/内容进行关键词查询，并展示素材标题、类型、内容、创建时间等列表信息，支持批量操作。

## 二、 顶部指标区

### 1. 业务规则

| 指标         | 计算口径                | 说明                                                 |
| :----------- | :---------------------- | :--------------------------------------------------- |
| **素材总数** | 文本素材数 + 图片素材数 | 统计整个素材库，不随当前类型、分组或关键词筛选变化。 |
| **文本素材** | 类型为文本的素材数量    | 只统计文本素材。                                     |
| **图片素材** | 类型为图片的素材数量    | 只统计图片素材。                                     |
| **素材分组** | 分组数量                | 默认分组也计入。                                     |

## 三、 分组管理区

### 1. 搜索分组

- 支持根据分组名称模糊匹配。

### 2. 新增/编辑分组

- **规则**：新增商户时，生成默认分组，不允许编辑、删除。
- **表单字段**：
  | 字段 | 类型 | 是否必填 | 校验/说明 |
  | :--- | :--- | :--- | :--- |
  | **分组名称** | 文本输入 | 是 | 1. 最长 20 个字符；<br>2. 不能与当前类型（文本/图片）下的已有分组重复，重复提示“分组名称已存在”。 |

- **保存时需校验**：
  - 分组名称不能为空，为空提示“请输入分组名称”。
  - 分组名称不能超过 20 个字符。
  - 分组名称不能与当前类型下的已有分组重复，新增/编辑时都需要做唯一性校验，重复提示“分组名称已存在”。

### 3. 删除分组

- 删除分组前需弹窗让用户二次确认：确认删除该分组吗？删除后该分组下的内容将会移动到默认分组。
- 删除分组后，将当前分组下的素材移动到默认分组。

## 四、 查询条件与结果列表

### 1. 查询条件

| 字段/操作    | 类型     | 说明                     |
| :----------- | :------- | :----------------------- |
| **素材标题** | 文本输入 | 支持按素材标题模糊查询。 |

### 2. 结果列表

| 列区域       | 说明                                         |
| :----------- | :------------------------------------------- |
| **勾选列**   | 支持当前页多选，用于批量移动分组和批量删除。 |
| **素材标题** | 展示文本素材标题，过长省略。                 |
| **素材内容** | 展示文本内容摘要，完整内容在编辑弹窗维护。   |
| **创建人**   | 展示创建该素材的账号。                       |
| **创建时间** | 展示素材首次创建时间。                       |
| **操作**     | 支持编辑、移动分组、删除；删除前需二次确认。 |

## 五、 操作按钮区

### 1. 添加文本素材

- **操作方式**：文本 Tab 下点击“添加素材”后，打开新增文本素材弹窗。
- **表单字段**：
  | 字段 | 类型 | 是否必填 | 校验/说明 |
  | :--- | :--- | :--- | :--- |
  | **素材标题** | 文本输入 | 是 | 最长 20 个字符；为空提示“请输入素材标题”。 |
  | **素材内容** | 多行文本输入 | 是 | 最长 2000 个字符；为空提示“请输入素材内容”。 |

- **保存时需校验**：
  - 素材标题、素材内容不能为空，否则提示“请输入XXX”。
  - 素材标题不能超过 20 个字符，素材内容不能超过 2000 个字符。
  - 保存成功后关闭弹窗，刷新当前文本列表。

### 2. 添加图片素材

- **操作方式**：图片 Tab 下点击“添加素材”后，直接打开系统文件选择器，支持多选上传。
- **表单字段**：
  | 字段 | 类型 | 是否必填 | 校验/说明 |
  | :--- | :--- | :--- | :--- |
  | **图片文件** | 文件选择 | 是 | 支持 jpg、jpeg、png、gif、webp；单张最大 10MB。 |
  | **素材名称** | 系统生成 | 否 | 最长 200 字符，默认取上传文件名；后续可在图片卡片标题处重命名。 |

- **保存时需校验**：
  - 未选择图片时，上传按钮置灰。
  - 图片格式或大小不符合要求时，不允许选择。
  - 上传成功后刷新当前图片列表。

### 3. 导入文本

- **操作方式**：仅文本 Tab 展示“导入文本”，图片 Tab 不展示。
- **支持格式**：支持导入 `.xls`、`.csv`，字段如下：
  | 字段 | 类型 | 是否必填 | 校验/说明 |
  | :--- | :--- | :--- | :--- |
  | **文本标题** | 模板第 1 列 | 是 | 标题 20 字以内 |
  | **文本内容** | 模板第 2 列 | 是 | 文本内容 2000 字以内 |

- **导入时需校验**：
  - 文件格式须符合要求。
  - 有效行的文本标题、文本内容不能为空；标题和内容都为空的行跳过。
  - 任一有效行校验失败时，本次导入不入库，支持下载失败原因。
  - 导入成功后刷新当前文本列表。

### 4. 移动分组/批量移动分组

- **操作说明**：批量移动分组时，至少选择 1 条素材；未选择时按钮禁用，鼠标悬浮时提示：“至少选中一项数据”。
- **字段说明**：
  | 字段 | 类型 | 是否必填 | 校验/说明 |
  | :--- | :--- | :--- | :--- |
  | **目标分组** | 下拉选择 | 是 | 只能选择当前素材类型下已有分组。 |

### 5. 删除素材/批量删除

- **操作说明**：
  | 操作 | 提示文案 | 提交结果 |
  | :--- | :--- | :--- |
  | **单条删除** | 确定删除该素材吗？删除后，该素材将从素材库中移除，并自动删除快捷回复、欢迎语、关键词回复中引用的素材内容。 | 删除当前素材，成功后刷新列表。 |
  | **批量删除** | 1. 至少选择 1 条素材；未选择时按钮禁用，鼠标悬浮时提示：“至少选中一项数据”。<br>2. 确定删除选中的 N 条素材吗？删除后，该素材将从素材库中移除，并自动删除快捷回复、欢迎语、关键词回复中引用的素材内容。 | 删除全部已选素材，成功后刷新列表并清空已选状态。 |
